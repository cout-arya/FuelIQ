═══════════════════════════════════════
PORTION CONTEXT ADJUSTMENTS
═══════════════════════════════════════

HOME COOKING: Base values (listed above)
RESTAURANT: Add 30-50% more calories (more oil, butter, cream)
DHABA: Add 40-60% more calories (heavy oil, ghee)
STREET FOOD: Add 20-40% more calories (deep frying)
DABBA/TIFFIN: Similar to home cooking (-5 to +10%)

COOKING METHOD CALORIE IMPACT:
- Steaming: No added calories
- Dry roasting: +5% calories
- Shallow fry (light oil): +15-20% calories
- Deep fry: +40-60% calories
- Tandoor: +5% calories (minimal oil)
- Gravy (tomato-onion): +50-80 kcal per serving
- Gravy (cream-based): +100-150 kcal per serving
